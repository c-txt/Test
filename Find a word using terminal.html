Find a word using terminal

#terminal #grep

To find a word or pattern in a file, use the basic grep syntax:

  • grep "string of text" filename

    • This command returns the full text of each line containing the text string you entered.
    • If you only want to see the matching words or strings, add the -o option.

-i option to ignore case

If you only want to see a result if it's a word on its own—not within a longer word—use the -w option.

If you want to know how many times certain text appears in a file, use the -c option.

To find lines that contain two specific words or patterns, you can use the pipe symbol to perform a second grep command. For example, let's say you want to find all lines that contain the words "wikiHow" and "Google:"

  • grep -i "wikiHow" filename.txt | grep -i "Google"

Another way to do this is by using regular expressions and the -E option

  • grep -E "wikiHow.*Google|Google.*wikiHow" filename.txt OR
grep -E 'wikiHow|Google' filename.txt

To search for a word across multiple files in the terminal, you can use the grep command with the -r option for recursive searching. For example, use grep -r "search_word" /path/to/directory/ to find "search_word" in all files within the specified directory and its subdirectories.

to find all instances of the word "wikiHow" in your Python files, but not in other files in a directory.  grep wikiHow *.py

to know the name of each file containing a certain word or series of characters, use the -l option.

grep -l "string of text" /home/myusername


* The grep command is case-sensitive, but you can use _grep -i_ to ignore case.
