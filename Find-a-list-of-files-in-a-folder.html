Find a list of files in a folder

#terminal #mac

If want the full paths of the regular files whose name ends in .lua in or under the /media/username directory, then you could use the standard find.

LC_ALL=C find /media/username -name '*.lua' -type f

Ref: https://unix.stackexchange.com/questions/762425/how-can-i-get-a-file-list-recursively
